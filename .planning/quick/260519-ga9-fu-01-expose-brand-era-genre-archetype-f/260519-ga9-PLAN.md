<objective>
FU-01 — add Brand / Era / Genre / Archetype controls to the `/search` Watches-tab
Filter drawer so the user can pick or change those facets in place, without a
round-trip to `/explore`.

Purpose: today brand/era/genre/archetype only arrive at `/search` as inline
removable chips on an Explore deep-link. The Filter drawer itself only offers
movement/size/style, so the only way to switch to a *different* facet is to go
back to `/explore` and re-pick. This closes that gap.

Output: four new chip-group components matching the existing
Movement/Size/Style control style, wired into `FilterDrawer` against the
existing `useSearchState` setters; a small server-side brand-facet fetch so the
brand control has option values.
</objective>
