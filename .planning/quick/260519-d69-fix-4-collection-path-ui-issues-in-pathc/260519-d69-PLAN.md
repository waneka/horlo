Do NOT add a fixed pixel height to the image; `aspect-square` driven by an equal-width parent is what guarantees consistency.
    4. The row container `<div className="hidden md:flex gap-4 items-start">` stays as-is — `gap-4` (16px) is the spec connector gap.
