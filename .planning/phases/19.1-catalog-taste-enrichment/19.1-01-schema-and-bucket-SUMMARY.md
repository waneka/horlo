## Storage Bucket

**Bucket name:** `catalog-source-photos`
**Settings:** private (signed URLs), 8 MB limit, allowed MIME: image/jpeg, image/png, image/webp

**4 RLS policies:**
1. `catalog_source_photos_select_authenticated` — FOR SELECT TO authenticated USING (bucket_id = 'catalog-source-photos')
2. `catalog_source_photos_insert_own_folder` — FOR INSERT TO authenticated WITH CHECK folder-enforcement
3. `catalog_source_photos_update_own_folder` — FOR UPDATE TO authenticated USING + WITH CHECK folder-enforcement
4. `catalog_source_photos_delete_own_folder` — FOR DELETE TO authenticated USING folder-enforcement

Folder-enforcement pattern: `(storage.foldername(name))[1] = (SELECT auth.uid())::text`
