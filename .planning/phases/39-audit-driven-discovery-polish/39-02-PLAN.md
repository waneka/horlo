### Load-bearing details

1. **Privacy split (Pitfall 1 + T-39-01):** TWO separate `if` statements. NEVER collapse `if (!overlap || !overlap.hasAny) notFound()` and the fallback Card into a single guard — that would cause anonymous viewers / private-collection viewers to see "No shared watches yet" instead of 404, leaking the existence of the tab.

2. **Apostrophes (Pitfall 4):** Use `&apos;` for EVERY apostrophe in JSX text content. Three occurrences in the body sentence: `don&apos;t`, `doesn&apos;t`, `don&apos;t`. One occurrence in the primary CTA label: `&apos;s collection →`. Raw `'` will fail ESLint `react/no-unescaped-entities`.

3. **Em-dash:** Use `—` (U+2014) verbatim, NOT two hyphens `--`. UI-SPEC § Copywriting Contract pins this.

4. **Right-arrow:** Use `→` (U+2192) verbatim at the END of each CTA label.

5. **displayName fallback (Pitfall 5):** Use `displayName ?? \`@${profile.username}\`` in the primary CTA label. `displayName` is `string | null` (computed at existing line 65 as `profile.displayName ?? null`). If null, the CTA must read `Browse @{profile.username}&apos;s collection →`.

6. **No `'use client'` directive:** page.tsx is a Server Component. Card + Link + buttonVariants composition is fully server-renderable. Adding `'use client'` would be scope drift (UI-SPEC § Server vs Client component constraint).

7. **Button primitive does NOT support `asChild`:** Use `buttonVariants(...)` className directly on `<Link>`. Do NOT import `Button` and try `<Button asChild>` — the project's Button wraps `@base-ui/react/button` (no Radix Slot). UI-SPEC § Button + Link composition pins this.

8. **Variants:** Primary CTA uses `variant: 'default'` (filled). Secondary CTA uses `variant: 'outline'`. Both use `size: 'default'`. UI-SPEC § Component Inventory locks this.

9. **CardContent className:** `space-y-4` (UI-SPEC § Spacing Scale md row). The interior CTA group is `flex flex-col gap-3 sm:flex-row sm:gap-4` — mobile-stacked, desktop-inline.

10. **`/explore` href is plain — no `#popular-collectors` anchor.** Per UI-SPEC § NSV-12 Copywriting Contract: "anchoring to `#popular-collectors` is allowed if the existing `/explore` page provides such an anchor; otherwise plain `/explore`." Do NOT add the anchor.
