case reversed), silently (D-02), without changing the DAL gate and without affecting any non-owner
viewer (GATE-03) or the owner's count badge (CMNT-09).

sees a compose box on their own watch ‚ÄĒ UAT flagged this (notes cover private context, not public
comments-to-self). SC-6 adds a route-level display gate: `canCommentDisplay = isOwner ? false :

(RESEARCH correction superseding the UI-SPEC) so the owner's badge still shows. No DAL or Server
Action changes; wears hosts never set `suppressCompose`.
