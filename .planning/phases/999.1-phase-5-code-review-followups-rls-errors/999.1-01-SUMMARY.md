## Decisions Made

- **Local mirror state stays on save failure (no rollback).** The review explicitly called rollback "optional"; `revalidatePath('/preferences')` in `savePreferences` runs only on success, so the next navigation naturally reconciles local state with server truth. A visible error banner is enough to warn the user; flipping the local toggle back would be jarring mid-edit.
- **MR-02 is import hygiene only.** The review's secondary suggestion ("distinguish Unauthorized vs infra errors via `err instanceof UnauthorizedError`") is a real improvement but a larger semantic shift — not in scope for this paperwork phase. The catch-all behavior is preserved verbatim.
- **MR-03 is closed via documentation.** Phase 6 enabled RLS; Phase 11 audited it. Re-adding RLS here would duplicate live policies and risk migration drift in prod. Closure note suffices.
