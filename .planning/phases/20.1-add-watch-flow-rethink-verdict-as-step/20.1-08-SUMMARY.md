## Decisions Made

- **Wrap manual-entry render branch in a `<div className="space-y-4">` fragment** — Places the Cancel back-link in natural reading order ABOVE the WatchForm. Users scanning top-down see the exit before committing to filling the form.
- **Reuse existing handleStartOver, do not create a new handler** — The helper already implements the correct idle-restore semantics (`setUrl('')` + `setState({ kind: 'idle' })`) and is already trusted by the extraction-failed branch's "Try another URL" button. Reusing it eliminates duplicate transition logic and keeps the state-machine surface unchanged.
- **Mirror PasteSection's "or enter manually" Tailwind classes exactly** — Symmetric entry/exit visual treatment lets users associate the two affordances at a glance.
- **Use `←` literal arrow character (not HTML entity)** — Plain Unicode in JSX text content is ESLint-clean and renders correctly across the supported font stack (IBM Plex Sans + Geist).
