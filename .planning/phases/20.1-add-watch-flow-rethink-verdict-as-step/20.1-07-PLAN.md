<!-- base-ui Accordion data-attribute contract (verified from node_modules) -->

From node_modules/@base-ui/react/accordion/panel/AccordionPanelDataAttributes.js:
```
AccordionPanelDataAttributes.index            = "data-index"            // current index (number value)
AccordionPanelDataAttributes.open             = "data-open"             // PRESENT WHEN OPEN (no value)
AccordionPanelDataAttributes.orientation      = "data-orientation"
AccordionPanelDataAttributes.disabled         = "data-disabled"
AccordionPanelDataAttributes.startingStyle    = "data-starting-style"   // animating in
AccordionPanelDataAttributes.endingStyle      = "data-ending-style"     // animating out
```

KEY POINT: `data-state` does NOT exist on base-ui's AccordionPanel. Only `data-open` (boolean attr).

Tailwind selector mapping:
- `data-[open]:foo`        applies when data-open attribute is present (panel open)
- `not-data-[open]:foo`    applies when data-open attribute is absent (panel closed)
- `data-[starting-style]:foo` applies during the open transition (mount)
- `data-[ending-style]:foo`   applies during the close transition (unmount)

CSS variable available for height transition:
- `--accordion-panel-height` (set on the panel element by base-ui)
