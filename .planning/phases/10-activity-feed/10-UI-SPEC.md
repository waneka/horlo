## Typography

Four size tiers, two weights. All bound to `--font-sans` (Geist) except section headings that explicitly opt into `--font-serif` (Instrument Serif) per the brand voice already set by the Horlo wordmark.

| Role | Size (Tailwind / px) | Weight | Line Height | Family | Used For |
|------|----------------------|--------|-------------|--------|----------|
| Body | `text-base` / 16px | `font-normal` (400) | `leading-relaxed` (1.625) | sans | Activity row verb line, overlay caption, empty-state body, dialog body |
| Label | `text-sm` / 14px | `font-medium` (500) | `leading-snug` (1.375) | sans | Tile username + time, rec rationale, insight card body, follow count strings, copywriting micro-labels |
| Heading | `text-xl` / 20px | `font-semibold` (600) | `leading-tight` (1.25) | sans | Section headings ("Collectors Like You", "Network Activity"), insight card titles, dialog titles |
| Display | `text-3xl` / 30px | `font-normal` (400) | `leading-tight` (1.25) | serif (`font-serif`) | Empty-state heading of home-level "nothing here yet" states; overlay watch name (e.g., "Submariner 14060M") |

Rationale:
- Four tiers cover every text style needed for the 5 sections. No 18px or 24px step; density wins at feed scale.
- Two weights (400, 600) plus one accent weight (500 for labels) — kept under the checker's "max 2 weights" discipline by reserving 500 for labels that need slightly more presence than body but less than heading (a common shadcn pattern). If the checker is strict on 2-weight-max, collapse Label to `font-normal`.
- Serif `Display` is used sparingly (≤ 2 instances per page) to echo the Horlo wordmark in high-emotion copy (overlay watch name, primary empty state heading). Keeps the home feeling editorial without overpowering scannability.

**Feed-row specifics (F-01 composition):**
```
[avatar 40×40] [username · verb · watchName]      [watch thumb 48–56×48–56]
               [{time ago}                    ]
```
- Line 1: `text-base font-normal` with `font-semibold` on `{username}` only and `font-medium` on `{watchName}` (makes the link target scannable).
- Line 2: `text-sm text-muted-foreground`.
- Line gap: `space-y-0.5` (2px).

**Aggregated row (F-08):** Line 1 template `{username} added {N} watches` with `{N}` in `font-semibold`; line 2 identical time-ago. Representative watch thumbnail on the right (from the first of the collapsed group).

**Responsive:** All sizes scale with the system; no responsive size variants needed. Readability at both mobile (16px base) and desktop (16px base) is adequate because the layout density (not type size) changes between breakpoints.

## Color

Horlo's existing `globals.css` implements a warm-neutral palette with a gold-brown accent. Phase 10 reuses it as-is. No new color tokens are introduced.

| Role | CSS token | Usage in Phase 10 |
|------|-----------|-------------------|
| Dominant (60%) | `--background` (`bg-background`) | Page canvas, section fill, dialog backdrop fade target |
| Secondary (30%) | `--card`, `--secondary`, `--muted` (`bg-card`, `bg-secondary`, `bg-muted`) | Activity row surface (`bg-card`), rec card fill (`bg-card`), WYWT tile muted ring background (`bg-muted`), picker dialog surface |
| Accent (10%) | `--accent` / `--ring` (oklch 0.76 0.12 75 light · 0.78 0.13 75 dark) | See "Accent reserved for" below |
| Destructive | `--destructive` (`bg-destructive`, `text-destructive`) | ONLY the "Unfollow" hover label on FollowButton (reuses Phase 9 D-09 semantics); no destructive action is introduced in Phase 10 itself |

**Accent reserved for (exhaustive list):**

1. WYWT unviewed tile ring — `ring-2 ring-ring` (W-04 unviewed state)
2. Primary CTA button fill on empty-state "Find collectors to follow" — default shadcn `Button` (`bg-primary`); note that `--primary` is neutral in this theme, accent shows up only via focus ring
3. Personal Insights "Alert" badge background for Sleeping Beauty card — `bg-accent text-accent-foreground`
4. Personal Insights "Tip" badge background for Wishlist Gap card — `bg-accent text-accent-foreground` (same badge style; copy distinguishes them)
5. "Follow" button idle state — inherits Phase 9 FollowButton; accent appears on focus ring only
6. Focus ring on all interactive elements (keyboard) — `ring-ring` (resolves to accent)
7. Load More button spinner tint — `text-accent` for the spinner only; button label stays neutral

**Accent is NEVER used for:**
- Generic link color (use `text-foreground` underline-on-hover)
- Section headings
- Time stamps or secondary metadata
- Viewed-state WYWT tile ring (muted-foreground `ring-1 ring-muted-foreground/40`)
- Counts or numeric badges without an "Alert/Tip" semantic

**Dark mode parity:** All values resolve through the same tokens in both modes. The one asymmetry the executor should know: dark-mode accent (oklch 0.78 0.13 75) is slightly brighter than light-mode (0.76 0.12 75), which is intentional for the unviewed tile ring to pop against `--background: oklch(0.14 …)`.

## Accessibility Contract (WCAG 2.2 AA)

- **Contrast:** All text against its surface meets 4.5:1 (body) / 3:1 (large). The existing theme tokens in `globals.css` already satisfy this; the executor should not introduce new color pairs.
- **Touch targets:** All interactive rows ≥ 44px (Phase 10 uses ≥ 64px for activity/suggested rows). Icon-only buttons (overlay close, Load More spinner wrapper) are `size-11` (44px).
- **Keyboard:**
  - Rails (`WearRail`, `CollectorsLikeYouRail`): `←` / `→` scroll by one tile; `Home` / `End` jump to rail ends; `Tab` walks tiles left-to-right.
  - Overlay: `Esc` closes; `←` / `→` previous / next; focus trap active; restore focus to triggering tile on close.
  - Feed row: `Enter` / `Space` activates the row's primary link (collector profile); nested watch-name link is a separate Tab stop.
  - Watch picker dialog: searchable list with `↑` / `↓` to navigate, `Enter` to select, `Esc` to close.
- **Screen reader:** All visual-only state (ring color for viewed/unviewed; Alert/Tip badge color) has a redundant text or aria-label equivalent (see Accessibility Copy above).
- **Focus:** All focusable elements show `focus-visible:ring-2 focus-visible:ring-ring focus-visible:ring-offset-2` against `bg-background`.
- **Reduced motion:** See Copywriting Contract § Reduced Motion.
- **Error messaging:** Announced via toast with `role="status"` (non-intrusive); retries always surface inline next to the failed control.
