interface DupeBannerProps {

}

- Headline: `<p className="text-sm font-semibold text-foreground">` ("Already in your collection" | "On your wishlist")
- Subtext (only when existingReference is non-null): `<p className="text-sm text-muted-foreground">Reference: {existingReference}</p>`
- Action row: `<div className="flex flex-col gap-2 sm:flex-row sm:gap-3">`
- Each Button: `className="w-full sm:flex-1"`, `type="button"`, `disabled={pending}`, `min-h-[44px]` WCAG touch target (combine with cn() if needed)

Conditional rendering:
- "View existing" Button: rendered ONLY when `existingReference !== null`
- "Move to Collection" Button: rendered ONLY when `existingStatus === 'wishlist'` AND `onMoveToCollection` is defined
- "Add another copy" Button: ALWAYS rendered (Button variant="outline")
- Loader2 swap inside "Move to Collection" Button when `pending === true`: `<><Loader2 className="size-4 mr-2 animate-spin" aria-hidden="true" /> Moving…</>` (U+2026 ellipsis verbatim)
