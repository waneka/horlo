## STRIDE Threat Register

|-----------|----------|-----------|-------------|-----------------|
| T-45-09 | Elevation of Privilege | direct HTTP POST to a curated-list Server Action by a non-owner | mitigate | `assertOwner()` is the first statement in every action — independent of the layout guard (D-06) |
| T-45-10 | Tampering | mass assignment via extra keys in action payload | mitigate | All zod schemas use `.strict()` — unknown keys rejected |
| T-45-11 | Tampering | XSS via markdown intro copy | accept | Content is owner-authored only; react-markdown blocks `javascript:` URLs by default; no `rehype-sanitize` needed for admin-only content (RESEARCH A2) |
