## Sampling Rate

- **After every task commit:** Run `npm run lint`
- **After every plan wave:** Run `npm run build` (full typecheck + Next.js build)
- **Before `/gsd-verify-work`:** `npm run build` green, then deploy to Vercel for visual checks
- **Max feedback latency:** ~90 seconds (build)
